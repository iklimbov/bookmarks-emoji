bookmarks + emoji

features + faq 

You can replace upto 512 icons with emojis because of the Chrome storage limitation.

Emoji is assigned to domain + path combination. That means, assigning emoji to a bookmark will assign the same emoji to bookmarks with similar URL.

Not all emojis are supported by all operating systems. If your emoji is not displayed properly on one of the computers simply replace emoji.

Customizing Bookmarks with Emoji

Replace default icons for bookmarks or folders with emoji. Right click icon to customize and select Emoji dialog. Alternatively, this option is available in the Edit dialog. Take your time selecting meaningful emojis. You can remove emoji in Emoji dialog or by clearing the preview box in Edit dialog.

Simple Tree

Bookmarks are organized into the tree with titles beside icons. Sort bookmarks by titles or number of uses.

Icons only

But what if you know your bookmarks? Do you really want to read titles every time? Sorting is not super intuitive in this view.

Tree vs Single Folder View

Tree view displays all the folders at the same time. Folder View includes navigator panel and shows only one folder at a time. To make the most out of single folder navigator, take your time and set custom emoji icons for all your folders. You will not regret the result!

Archiving

All bookmarks not used in the past 90 days are moved to an Archive folder, located in Other Bookmarks. Review them and get rid of them if you wish.

Deleting

Deleted bookmarks are moved into Recycling Bin. Deleting from Recycling Bin cannot be undone. Recycling Bin is located under Other Bookmarks.

More...

Switch the lights off to use dark mode. Use search to quickly find bookmarks and folders.
